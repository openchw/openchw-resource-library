# OpenCHW Resource Library

Public static library for final, approved OpenCHW PDFs and program materials.

## Operating model

- GitHub Pages hosts the public site.
- GoDaddy remains the domain registrar and DNS manager.
- `openchw.org` points to this GitHub Pages site.
- Final approved PDFs are uploaded into predictable folders.
- Public PDF URLs are used in OpenCHW CSV files and public resource content.

## Folder structure

```text
materials/
  sessions/
  resource-directories/
  forms/
  grants/
  program-guides/
  images/
```

## URL examples

```text
https://openchw.org/materials/sessions/session-1-caregiver-orientation.pdf
https://openchw.org/materials/resource-directories/sumner-county-caregiver-resources.pdf
https://openchw.org/materials/forms/caregiver-intake-preview.pdf
```

## Publication rule

Only upload materials approved for public viewing. Do not upload private caregiver data, protected health information, credentials, draft files, or internal-only documents.
